## 📌 Project Description

본 프로젝트는 자료구조 수업의 일환으로 진행되는 프로젝트입니다.  
추후 확정된 주제에 따라 구조체, 배열, API 등 다양한 기술을 활용한 프로그램을 구현할 예정입니다.

## 🔧 Technologies To Be Used

- Struct (`typedef struct`)
- Array and Dynamic Memory Allocation
- 기타 자료구조 및 알고리즘 (추후 확정)
